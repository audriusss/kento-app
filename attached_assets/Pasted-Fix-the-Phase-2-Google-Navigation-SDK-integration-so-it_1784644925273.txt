1. Audio guidance

Navigator.AudioGuidance

2. Navigator initialization

Use:

NavigationApi.getNavigator(
    activity,
    object : NavigationApi.NavigatorListener {

        override fun onError(
            @NavigationApi.ErrorCode errorCode: Int
